Portfolio

Each project in my portfoliio was built in order to learn new skills reinforece previously learned concepts. Areas that I focused on were:

  • Programming logic and Secure Authorization
  • Incorporating multiple "front-end" languages
  • Utilizing web libraries
  • Building relational databases
  • Application program interfaces
  • Front Desk This app was built to service the Malaysian real estate industry by automating most tasks related to managing a personal property. It was built using Ruby on Rails, PostgreSQL, HTML, Javascript, CSS and features multiple third-party API's
  • Free Parking This app was developed to help drivers find parking spaces in urban environments. It was built using Ruby on Rails, PostgreSQL, HTML, Javascript, CSS and features multiple third-party API's
  • Bitly Clone This app was built to gain experience in logic-based programming and databases. Like the real Bitly, it will shorten any url to a new random nine character url. It was built using Sinatra, PostgreSQL and vanilla HTML, CSS, JS and AJAX.
  • Quora Clone This clone of Quora was built to learn relational databases. It was built using Sinatra, PostgreSQL and vanilla HTML, CSS, JS.
  • SailingWithAndy This site was built using Wordpress. It utilizes various WP plug-ins was used to document and feature my videos of my journey sailing around the world.
